## 1. 프로젝트 개요

군위몰입 수학·영어의 선생님과 학생들이 직접 기획하고 코딩한 교육용 웹 앱을 한 곳에서 공유·체험하는 **앱 갤러리 플랫폼**입니다.

### 구현 현황

| 항목 | 상태 |
|---|---|
| 앱 갤러리 (등록/수정/삭제/공유) | ✅ 완료 |
| 강사/학생 역할 구분 인증 | ✅ 완료 |
| 커뮤니티 게시판 (댓글/핀 고정) | ✅ 완료 |
| 내 활동 탭 | ✅ 완료 |
| 모바일 반응형 | ✅ 완료 |
| Supabase 백엔드 5개 테이블 | ✅ 완료 |

## 2. 화면 구성

### 2.1 헤더

| 요소 | 설명 |
|---|---|

## 3. 기능 상세

### 3.1 앱 갤러리

- **무한 스크롤**: 10개 단위 로드, IntersectionObserver
- **서버 사이드 검색**: `title/description/author_name` ilike 필터 + 300ms 디바운스
- **필터**: 카테고리 5종 + 역할(강사/학생) + 정렬 3종
- **좋아요**: RPC fallback UPDATE + 인메모리 즉시 반영
- **조회수**: sessionStorage 중복 방지

### 3.2 앱 관리

| 기능 | 권한 | 설명 |
|---|---|---|
| 등록 | 로그인 누구나 | 표지(3MB) + HTML(5MB) 또는 URL |
| 수정 | 작성자 본인 | 기존 내용 자동 채움, 파일 미변경 시 유지 |
| 삭제 | 작성자 본인 | 확인 팝업 후 즉시 제거 |
| 공유 | 누구나 | `?app=UUID` URL 클립보드 복사 + 토스트 알림 |
| URL 직접 오픈 | 누구나 | 공유 URL 접속 시 해당 앱 자동 오픈 |

### 3.3 댓글 (앱)

- 로그인 사용자만 작성
- 본인 댓글: 호버 시 × 삭제 버튼

## 5. 인증 시스템

- 회원가입: 강사/학생 라디오 선택 → 강사 선택 시 실명 안내 박스
- 비밀번호 최소 6자
- Email Confirm 비활성화 (즉시 로그인)
- `handle_new_user` 트리거: username + role 자동 저장
- 헤더: "이름 강사" / "이름 학생" 표시

## 6. 디자인 시스템

### 6.1 색상

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg-base` | `#f1f5f9` | 배경 |
| `--bg-surface` | `#ffffff` | 카드 |
| `--primary` | `#6366f1` | 강조색 (인디고) |
| `--text-primary` | `#111827` | 본문 |

### 6.2 플로팅 UI 요소

- 레이어드 그림자: ambient + key shadow 2중 구조
- 카드 호버: -6px 상승 + 상단 shimmer 라인
- 헤더: saturate 높은 frosted glass
- 모달: 3중 그림자 + 내부 하이라이트 링

### 6.3 반응형

| 브레이크포인트 | 적용 내용 |
|---|---|
| `max-width: 900px` | 상세 모달 1컬럼, 댓글 영역 축소 |
| `max-width: 600px` | 1열 카드, 필터 왼쪽 정렬 스택, 로고 축소 |

## 8. Supabase 백엔드

### 8.1 테이블 구조

| 테이블 | 주요 컬럼 |
|---|---|
| `profiles` | id, username, role |
| `apps` | id, title, category, description, cover_image_url, app_url, html_file_path, author_id, author_name, author_role, views_count, likes_count, created_at |
| `comments` | id, app_id, author_id, author_name, content, created_at |
| `posts` | id, category, title, content, author_id, author_name, author_role, **is_pinned**, created_at |
| `post_comments` | id, post_id, author_id, author_name, author_role, content, created_at |

### 8.2 RPC 함수

| 함수 | 설명 |
|---|---|
| `toggle_post_pin(post_id, pin_value)` | 강사 역할 확인 후 is_pinned 토글 |
| `increment_views(row_id)` | 조회수 원자적 증가 (선택) |
| `increment_likes(row_id)` | 좋아요 원자적 증가 (선택) |

### 8.3 Storage

- 버킷: `app-gallery` (Public)
- 경로: `covers/`, `htmls/`
